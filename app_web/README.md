# XunKong 总后台

> 这是用 vue + vite + arco-design 搭建的 总管理员后台

### 技术文档
Vue:   https://cn.vuejs.org/guide/introduction

Vite: https://cn.vitejs.dev/

ArcoDesignVue: https://arco.design/vue/docs/start


## 1. 启动说明

###  初次启动安装依赖
```
 npm i
```

###  启动项目

```

 npm run dev

```

###  打包项目

```

 npm run build

```
